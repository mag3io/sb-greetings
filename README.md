# SandBox Greetings

## Overview
This example uses the [expressjs](http://expressjs.com/) framework.


## Running the server
To run the server, follow these simple steps:

```
cd server
npm install
npm start
```

REST API:

```
GET http://localhost:3000/greetings/hello?name=
```

```
GET http://localhost:3000/greetings/bye?name=
```
